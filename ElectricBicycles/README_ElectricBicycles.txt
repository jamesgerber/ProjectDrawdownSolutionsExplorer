Notes for Electric Bicycles solution

This solution hinges on the assumption that an electric bicycle 
replaces 688 pkm of car travel; calculation is carried out for
adoption scenarios in terms of number of cars (44%, 80%)

Important note:  Only mapping private bicycles.  

Ebike uses 0.00932 kwh per km, per the assessment document


Main code is Ebikecalcs.m

This runs some codes that were developed for Electric Cars.
